## Etat d'avancement actuel :

- Projet terminé :
  - Toutes les pages ont été codées. (HTML + CSS)
  - Responsif fonctionnel sur les 3 formats d'écran
  - Darkmode fonctionnel en fonction des préférences du systeme de l'utilisateur
  - La navigation TAB est fonctionnelle et visible


- Difficultés rencontrées :
	- Définir a l'avance l'organisation de la page en mobile pour qu'elle s'adapte facilement par la suite en desktop.
	- Gérer un design systeme depuis la maquette PDF pour gerer les elements communs. (En regardant juste la version PDF on a l'impression que certains titres ou paragraph d'une page a l'autre ne sont pas forcement de la meme taille alors qu'ils sont dans une card de meme apparance que sur d'autres page..) j'ai donc géré au cas par cas.

- Methode de travail : 
	- J'ai d'abord analysé les differentes maquettes afin de lister les elements de style réccurents (boutons, cards, couleurs, taille de texte etc...)
	- J'ai ensuite analysé le comportement du header et footer communs a toutes les pages sur les differents agencements afin de realiser un gabarit qui serviera de modele aux autre pages.
	- J'ai ensuite posé les bases de mon css avec des variables de base (mode clair) et des valeurs alternatives pour le mode sombre. Ainsi que les elements communs (cards, boutons, tags etc..)
	- J'ai ensuite travaillé page par page, HTML d'abord, css ensuite. Cela semblait plus facile au fil du temps. (Logique de conception entre les differents agencements - Gestion des grid)

- Avec un peu plus de temps : 
	- Mise en place des animations au survol.
	- Meilleur refacto du css (certaines redondances..).
	- Je passerai plus de temps sur l'accessibilité.
